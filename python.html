What You'll Learn

Python Fundamentals

Grasp core concepts like variables, data types (numbers, strings, lists, tuples, dictionaries), operators, and control flow.

Functions & Modules

Learn to write reusable code with functions and organize your projects using modules and packages.

Object-Oriented Programming (OOP)

Understand the principles of OOP in Python: classes, objects, inheritance, and polymorphism.

Data Structures

Work effectively with Python's built-in data structures like lists, tuples, sets, and dictionaries.

File Handling

Learn how to read data from various file types (text, CSV) and write data back to files.

Standard Library & PIP

Explore useful modules in Python's standard library and learn how to install external packages using pip.

Course Curriculum

Module 1: Python Fundamentals
Introduction, Setup, Variables, Data Types (Numbers, Strings), Basic Operators, Input/Output.
Module 2: Data Structures
Lists, Tuples, Dictionaries, Sets, String Manipulation.
Module 3: Control Flow & Functions
Conditional Statements (if/elif/else), Loops (for, while), Defining Functions, Arguments, Scope.
Module 4: Object-Oriented Programming (OOP)
Introduction to OOP, Classes & Objects, Inheritance, Encapsulation, Polymorphism.
Module 5: Modules, Packages & File Handling
Importing Modules, Standard Library Overview, Installing Packages with PIP, Reading and Writing Files.

Prerequisites

  • Basic computer skills (navigating files, using a text editor).
  • Logical thinking and an eagerness to learn programming concepts.
  • No prior programming experience is necessary.
  • Access to a computer with Python 3 installed.
