1.1 Introduction & Setup (JDK/IDE)

Understand Java's platform independence (Write Once, Run Anywhere) and set up the Java Development Kit (JDK) and an Integrated Development Environment (IDE) like IntelliJ IDEA or Eclipse.

Study Materials

  • W3Schools: Java Syntax

  • Oracle Java Tutorials: "Hello World!" Application

Learn about Java's primitive data types (int, float, double, boolean, char, etc.) and reference types (like String), and how to declare variables.

Study Materials

  • W3Schools: Java Variables
  • W3Schools: Java Data Types
  • Oracle Java Tutorials: Primitive Data Types

Understand arithmetic, relational, logical, assignment, and other operators in Java.

Study Materials

  • W3Schools: Java Operators
