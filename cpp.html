Build a strong foundation in C++, a powerful language used in game development, systems programming, and high-performance computing.

What You'll Learn

C++ Fundamentals

Master core concepts like variables, data types, operators, control flow (loops, conditionals), and functions.

Object-Oriented Programming (OOP)

Understand classes, objects, inheritance, polymorphism, encapsulation, and abstraction in C++.

Memory Management

Learn about pointers, references, dynamic memory allocation (new/delete), and smart pointers.

Data Structures & STL

Work with fundamental data structures (arrays, vectors, lists, maps) and the Standard Template Library (STL).

File I/O

Learn how to read from and write to files using C++ stream classes.

Error Handling

Understand exception handling mechanisms in C++ using try, catch, and throw.

Course Curriculum

Module 1: Introduction to C++
Setting up the environment, Basic Syntax, Variables & Data Types, Operators, Input/Output.
Module 2: Control Flow & Functions
Conditional Statements (if, else, switch), Loops (for, while, do-while), Functions, Scope.
Module 3: Pointers, Arrays & Strings
Understanding Pointers, Memory Addresses, Arrays, C-style Strings, C++ String Class.
Module 4: Object-Oriented Programming (OOP)
Classes & Objects, Constructors & Destructors, Inheritance, Polymorphism, Encapsulation.
Module 5: Advanced Topics & STL
Templates, Exception Handling, File I/O, Introduction to the Standard Template Library (Vectors, Maps, Algorithms).

Prerequisites

  • Basic computer literacy.
  • Logical thinking and problem-solving aptitude.
  • No prior programming experience is required, but it can be helpful.
  • Access to a C++ compiler (e.g., g++, Clang, MSVC).
