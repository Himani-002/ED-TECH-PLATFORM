What You'll Learn

  • Build responsive websites using HTML5 and CSS3.
  • Master JavaScript fundamentals and modern ES6+ features.
  • Develop dynamic front-end applications with React (or specify another framework).

Course Curriculum

  • How the web works: HTTP, Browsers, Servers
  • Introduction to HTML: Structure, Elements, Semantics
  • Introduction to CSS: Selectors, Box Model, Layouts
  • Developer Tools Basics
  • Variables, Data Types, Operators
  • Control Flow: Conditionals and Loops
  • Functions and Scope
  • DOM Manipulation and Events
  • Introduction to ES6+ Features
  • Introduction to React: Components, JSX, Props
  • State Management and Lifecycle
  • Hooks (useState, useEffect)
  • Routing with React Router
  • Fetching Data from APIs
  • Introduction to Node.js
  • Building REST APIs with Express
  • Middleware
  • Working with Databases (e.g., MongoDB with Mongoose)
  • Authentication and Authorization
  • Git Basics and GitHub Workflow
  • Introduction to Deployment (e.g., Heroku, Netlify, Vercel)
  • Final Project Development
