Final Project

Combine all the skills you've learned to build a complete full-stack web application. This is your chance to showcase your abilities by creating a project like a blog, an e-commerce site, or a social media platform.

Requirements (Example)

  • Must include both frontend (React) and backend (Node/Express) components.
  • Must use a database (MongoDB) for data persistence.
  • Must implement user authentication.
  • Must be deployed to a live hosting platform.
  • Code must be version controlled using Git and hosted on GitHub.
